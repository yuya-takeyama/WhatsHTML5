Agenda ====== - HTML5 とは何か - 何故 HTML5 を学ぶか - HTML5 デモ
What's HTML5 ? ============== 広義の HTML5 ------------ - 新しいタグ - デザイン的なもの (CSS3) - プログラミング的なもの
新しいタグ ========== 意味付け的なもの ---------------- - <header> - <nav> - <article> - 地味なので省略. (だけど実は大事)
新しいタグ ========== 機能的な要素 ------------ - <audio> - <video> - <canvas> - Flash を置き換えることができる.
新しいタグ ========== 逆に無くなったもの ------------------ - <center> - <font> - その他 CSS で表現すべきもの. - HTML は構造化, 意味付け, 文書の記述だけ.
デザイン的なもの (CSS3) ======================= 表現力の大幅な向上 ------------------ - 角の丸み - 影 - グラデーション - アニメーション
プログラミング的なもの ====================== - File API - Web Storage - 組込みデータベース
プログラミング的なもの ====================== デバイスとの連携 ---------------- - 位置情報 - マイク - カメラ
プログラミング的なもの ====================== ネットワークとの連携 -------------------- - XMLHTTPRequest Level2 - JSON - Web Socket - クライアント/サーバ双方向のやり取り - Ajax ではできなかったことの実現 - HTML5 とは別プロジェクトに
何故 HTML5 を学ぶか =================== HTML5 でできること ------------------ - スマートフォン最適化サイト - スマートフォンアプリ開発にも - PC 向けサイトは数年後 - ガラケー (笑)
スマートフォンサイト ==================== - CSS3 でスマートフォンぽいデザイン - JavaScript もゴリゴリ使える - ハードウェア的な限界 - ネットワーク的な限界
制約との戦い ============ ガラケーほどでは無いが... ------------------------- - 画像よりも CSS でデザインする - HTML/CSS でできることを JavaScript でやらない - 読み込むファイル数は少なく - 機能を詰め込まない勇気
アプリ開発と HTML5 ======================== - 手を抜けるポイントは WebView で - 保守しやすい - iPhone/Android 開発の共通化
